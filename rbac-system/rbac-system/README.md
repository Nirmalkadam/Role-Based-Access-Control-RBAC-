# RBAC System

## Description

A simple Node.js-based Role-Based Access Control (RBAC) system with secure authentication and authorization.

## Features

1. User Registration and Login
2. Role-Based Access Control (Admin, User, Moderator)
3. Secure Password Hashing and JWT Authentication
4. Protected Routes based on Roles

## Setup Instructions

1. Clone the repository.
2. Install dependencies: `npm install`.
3. Start the server: `nodemon server.js`.
4. Test endpoints using Postman or any REST client.

Enjoy secure access control for  application!
